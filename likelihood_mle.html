What is Likelihood?

Continuous Variables

For a discrete random variable, the probability of \(X\) taking a specific value \(x\) is

\[ P(X = x) \]

For a continuous random variable, the probability that \(X\) equals exactly one value is

\[ P(X = x) = 0 \]

Instead, we describe probabilities over an interval:

\[ P(x_1 \leq X \leq x_2) \]

To compute such probabilities, we introduce the Probability Density Function (pdf) \(f(x)\), which satisfies:

  1. For any interval \([x_1, x_2]\):
    \[ P(x_1 \leq X \leq x_2) = \int_{x_1}^{x_2} f(x)\,dx \]
  2. For all \(x\), \(f(x) \geq 0\), and
    \[ \int_{-\infty}^{\infty} f(x)\,dx = 1 \]

Suppose we observe \(n\) independent samples \(x_1, x_2, \dots, x_n\) from the same distribution. The joint density is

\[ P(X_1 = x_1, \dots, X_n = x_n) \propto \prod_{i=1}^n f(x_i) \]

This product is called the Likelihood Function:

\[ L(\theta) = \prod_{i=1}^n f(x_i \mid \theta) \]
