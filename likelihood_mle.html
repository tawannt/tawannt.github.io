What is Likelihood and MLE?

From probability density functions to the likelihood function and maximum likelihood estimation.

On this page

  1. Continuous Variables
  2. Likelihood Function
  3. Meaning of Likelihood
  4. Maximum Likelihood Estimation
Section 1

Continuous Variables

For a discrete random variable, we can represent the probability of \(X\) taking a specific value \(x\) as

\[ P(X = x), \]

which can take any value in \([0,1]\).

However, for a continuous random variable, the probability that \(X\) equals exactly one point is essentially zero:

\[ P(X = x) = 0. \]

Instead, we describe probabilities over an interval:

\[ P(x_1 \leq X \leq x_2). \]

Probability Density Function

To compute such probabilities, we introduce the Probability Density Function (pdf) \(f(x)\), which satisfies the following properties:

  1. For any interval \([x_1, x_2]\),
    \[ P(x_1 \leq X \leq x_2) = \int_{x_1}^{x_2} f(x)\,dx. \]
  2. For all \(x\), \(f(x) \geq 0\), and
    \[ \int_{-\infty}^{\infty} f(x)\,dx = 1. \]

In other words, the probability of \(X\) falling within an interval is given by the area under the curve of \(f(x)\) across that interval.

Since \(f(x) \geq 0\) everywhere, this interpretation is always consistent: if \(f(x)\) is small, the corresponding area (and hence probability) will also be small, and if \(f(x)\) is large, the area will be larger.

Section 2

The Likelihood Function

Suppose we observe \(n\) independent samples \(x_1, x_2, \dots, x_n\) from the same distribution with pdf \(f(x)\). The joint probability density of these samples is

\[ P(X_1 = x_1, \dots, X_n = x_n) \propto \prod_{i=1}^n f(x_i). \]

This product is called the Likelihood Function, and it plays a central role in parameter estimation:

\[ L(\theta) = \prod_{i=1}^n f(x_i \mid \theta), \]

where \(\theta\) denotes the parameter(s) of the pdf \(f\).

The likelihood \(L(\theta)\) is not a probability over \(\theta\) — it measures how probable the observed data is, given a particular value of \(\theta\).

Section 3

The Meaning of the Likelihood Function

Given observed data, \(L(\theta)\) tells us how consistent a parameter value \(\theta\) is with what we observed. A higher \(L(\theta)\) means the observed data is more probable under that choice of \(\theta\).

This shifts the question from "what is the probability of this data?" to "which parameter \(\theta\) makes this data most probable?"

Section 4

Maximum Likelihood Estimation

Maximum Likelihood Estimation (MLE) finds the parameter \(\hat{\theta}\) that maximizes the likelihood function:

\[ \hat{\theta} = \arg\max_{\theta}\; L(\theta) = \arg\max_{\theta} \prod_{i=1}^n f(x_i \mid \theta). \]

Log-Likelihood

Because products of many small numbers can underflow numerically, we typically maximize the log-likelihood instead (since \(\log\) is monotonically increasing, the maximizer is unchanged):

\[ \ell(\theta) = \log L(\theta) = \sum_{i=1}^n \log f(x_i \mid \theta). \]

MLE is one of the most widely used parameter estimation methods in statistics and machine learning, underpinning techniques from logistic regression to neural network training (cross-entropy loss is equivalent to negative log-likelihood under a categorical distribution).
